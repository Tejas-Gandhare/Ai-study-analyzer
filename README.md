# Ai-study-analyzer

The AI Study Analyzer is a data-driven application designed to analyze student habits and predict academic performance. It uses Machine Learning to estimate marks based on key factors such as study hours, sleep, phone usage, and attendance.

Unlike traditional prediction models, this project also provides personalized suggestions to help students improve their performance. The system combines predictive analytics with decision-making logic to deliver meaningful insights.

This project demonstrates the practical application of data science in solving real-world problems and highlights the importance of data-driven decision making in education.
